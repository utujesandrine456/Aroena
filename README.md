## рҹҡҖ Getting Started

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) `v18+`
- [Expo CLI](https://docs.expo.dev/get-started/installation/) `npm install -g expo-cli`
- [MongoDB](https://www.mongodb.com/) (local or Atlas)

<br/>

### 1 В· Clone the Repository

```bash
git clone https://github.com/utujesandrine456/Aroena.git
cd Aroena
```

### 2 В· Install Dependencies

```bash
npm install
```

### 3 В· Configure Environment

```bash
cp .env.example .env
# Fill in your MongoDB URI, Cloudinary keys, Flutterwave keys, Firebase config
```

### 4 В· Run the Mobile App

```bash
expo start
```

### 5 В· Run the Admin Web Portal

```bash
cd admin
npm start
```
